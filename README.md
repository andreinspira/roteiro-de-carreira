**O QUE FAZER:**

**ENTREGÁVEIS:**

**CRITÉRIOS DE ACEITAÇÃO:**  
- Accuracy acima de 70%  
- Código organizado e comentado

**DICA:** Use datasets públicos como IMDb Reviews ou Twitter Sentiment.

## 💬 ROTEIRO DE ENTREVISTAS  

**PERGUNTA 1:** O que é overfitting?  
**COMO RESPONDER:** Explique que é quando o modelo aprende demais os dados de treino e não generaliza bem para novos dados.  

**PERGUNTA 2:** Qual a diferença entre regressão e classificação?  
**COMO RESPONDER:** Regressão prevê valores contínuos, classificação prevê categorias.  

**PERGUNTA 3:** O que são features em Machine Learning?  
**COMO RESPONDER:** São as variáveis de entrada que alimentam o modelo.  

**PERGUNTA 4:** Cite uma métrica usada em classificação.  
**COMO RESPONDER:** Accuracy, precision, recall ou F1-score, explicando brevemente cada uma.  

**PERGUNTA 5:** Como você lidaria com dados desbalanceados?  
**COMO RESPONDER:** Usaria técnicas como oversampling, undersampling ou ajuste de pesos no modelo.
